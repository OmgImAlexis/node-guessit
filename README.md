## Install

[npm][]:

```sh
npm install node-guessit
```

[yarn][]:

```sh
yarn add node-guessit
```


## Usage

```js
const NodeGuessit = require('node-guessit');

const nodeGuessit = new NodeGuessit();

console.log(nodeGuessit.renderName());
// script
```
